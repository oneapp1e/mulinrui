搭建hexo+github+next博客

发表于 2018-01-19 | 分类于 hexo

搭建博客

为什么选择hexo呢，因为要拥抱markdown啊，workPress虽然也支持markdown，但是太不好用了。（hexo是一个基于Nodejs的静态博客网站生成器）

准备工作

  1. 创建GitHub 仓库

    登录注册

    github作为一个程序化必须必知道的一个网站，有账号的登录，没账号的赶紧注册吧。

    创建仓库

    注意仓库名称的地方一定要使用：你的用户名.github.io(上面是因为我已经创建过了的提示)，否则github默认不会给你生成GitHub Pages ，需要去创建好的仓库的设置里面更改。


    可以更改选择master branch，这样你的博客网址就是二级网址：http://oneapp1e.github.io/你的仓库名称


    只有创建的仓库名是：你的用户名.github.io，才是正解，可以到设置中更改名称啊！！

    总结

    在github上创建一个你的用户名.github.io 的仓库。

  2. 电脑git环境准备

    打开git官网，根据自己的系统下载进行安装。直接默认安装即可。

  3. Nodejs安装

    打开Nodejs官网，选择最新版本就行，没有特殊要求，直接默认安装即可。
